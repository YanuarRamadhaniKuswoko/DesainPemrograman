position: static;

Position static digunakan untuk mengatur elemen menjadi statis secara default. Elemen akan mengikuti posisi normal, tidak dipengaruhi oleh properti seperti top, bottom, left, dan right.

Elemen ini diatur menggunakan posisi: static;




position: relative;

Elemen HTML yang menggunakan position relative akan terletak pada posisi normal. Mengatur properti atas, kanan, bawah, dan kiri dari elemen relatif akan membuatnya bergeser dari posisi normalnya. Konten lain tidak akan disesuaikan untuk mengisi celah yang ditinggalkan.

Elemen ini diatur menggunakan posisi: relative;



position: fixed;

Elemen HTML yang menggunakan position fixed akan memiliki sifat tetap, bahkan jika halaman di-scroll. Properti left, bottom, top, dan right tetap berlaku.

Elemen ini diatur menggunakan posisi: fixed;










position: absolute;

Elemen HTML dengan position absolute akan diposisikan relatif terhadap elemen induk terdekat yang memiliki posisi selain static, bukan relatif terhadap layar secara normal.

Elemen ini diatur menggunakan posisi: relative;
Elemen ini diatur menggunakan posisi: absolute;

Catatan: IE/Edge 15 dan versi yang lebih lama tidak mendukung sticky.
